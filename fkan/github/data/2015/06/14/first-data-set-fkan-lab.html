The First Dataset on FKAN Lab

June 14, 2015

The Data project on FKAN Lab now has its first contribution.

This dataset - Kolar Schools - consists of Fluoride levels in water tested in schools in an informal survey in Kolar, Karnataka.

Thanks to Priyanka Jain for collecting this data and making it available under an open license.
